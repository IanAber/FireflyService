ElektrikGreen FireflyIO API

/ws GET start a web socket connection
/setRelay/{relay}/{on} PUT Turn on or off (1,0,on,off,true,false) the relay by name or number
/setOutput/{output}/{on} PUT Turn on or off (1,0,on,off,true,false) the digital output by name or number
/getStatus GET Returns a JSON object containing the full system status
/getFuelCell GET Returns the current status of the fuel cell only
/setFuelCell/TargetPower/{power} PUT Set the target power output
/setFuelCell/TargetBattHigh/{volts} PUT Set the battery high voltage setpoint
/setFuelCell/TargetBattLow/{volts} PUT Set the batery low voltage set point
/setFuelCell/Start PUT Start the fuel cell
/setFuelCell/Stop PUT Stop the fuel cell
/setFuelCell/ExhaustOpen PUT Start the water pump on high and begin air removal
/setFuelCell/ExhaustClose PUT Stop the exhaust function
/setFuelCell/Enable PUT Enable CAN communications to the fuel cell (we are always listening but may not be sending)
/setFuelCell/Disable PUT Disable CAN communications to the fuel cell so it can be controlled locally by its own user interface
/setFuelCell/ResetFault PUT Sends the fault reset message to the fuel cell
/getElectrolyser/{electrolyser} GET Returns the status of one or all electrolysers by name (all for all electrolysers)
/setElectrolyser/Production/{electrolyser}/{rate} PUT Set the production rate (60-100) by electrolyser name
/setElectrolyser/Start/{electrolyser} PUT Start the electrolyser by name
/setElectrolyser/Stop/{electrolyser} PUT Stop the electrolyser by name
/setElectrolyser/Preheat/{electrolyser} PUT Start the preheat cycle
/setElectrolyser/Reboot/{electrolyser PUT Reboot the electrolyser
/setElectrolyser/Locate/{electrolyser} PUT Start the locate cycle where the lights on the device flash to identify the machine
/setElectrolyser/Blowdown/{electrolyser} PUT Start the blowdown procedure
/setElectrolyser/Refill/{electrolyser} PUT Start the water refil process
/setElectrolyser/StartMaintenance/{electrolyser} PUT Start maintenance mode
/setElectrolyser/StopMaintenance/{electrolyser} PUT Stop maintenance mode
/setDryer/Start PUT Start the dryer
/setDryer/Stop PUT Stop the dryer
/setDryer/Reboot PUT Reboot the dryer
/getElectrolyser/production/{electrolyser} GET Returns the current production rate setting
/getElectrolyser/state/{electrolyser} GET Returns the electrolyser run state
/FuelCellData/DCDC?start={date/time}&end={date/time} GET Returns the JSON formatted data for the fuel cell DC to DC converter
/FuelCellData/Stack?start={date/time}&end={date/time} GET Returns the JSON formatted data for the fuel cell stack
/FuelCellData/Pressures?start={date/time}&end={date/time} GET Returns the JSON formatted data for the fuel cell gas pressures
/FuelCellData/Coolant?start={date/time}&end={date/time} GET Returns the JSON formatted data for the fuel cell coolant system
